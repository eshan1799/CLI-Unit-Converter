# CLI Unit Converter

## Description

A CLI application that converts centimetres into metres and vice versa!

## Installation

* ```npm i -g @eshan1799/unitconverter```

#### Prerequisites
* [Node.js](https://nodejs.org/en/)
* [npm](https://www.npmjs.com/get-npm)

## Usage

* ```converter "'enter string here'" ```

* String format: 
```x unit1 in unit2```

#### Example
* ```converter "5 m in cm" ```
--> output: 
```"500 cm"```
